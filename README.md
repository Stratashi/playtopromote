# PlayToPromote - Portal Panic

Multiplayer gaming platform where winners get their promotional links shared!

## Features
- ?? Multiplayer battles with Jeff the Robot Wizard
- ?? Portal mechanics and spell casting
- ?? 9 different enemy types including bosses
- ? Power-ups and dynamic difficulty
- ?? Winner promotion system
- ?? Mobile touch controls

## Setup

1. Clone the repository:
\`\`\`bash
git clone https://github.com/yourusername/portal-panic-game.git
cd portal-panic-game
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

3. Run development server:
\`\`\`bash
npm run dev
\`\`\`

4. Build for production:
\`\`\`bash
npm run build
\`\`\`

## Technologies
- React 18
- Vite
- TailwindCSS
- React Router
- Framer Motion
- Canvas API for game rendering

## Game Controls
- **WASD / Arrow Keys** - Move
- **Left Click** - Shoot spells
- **Right Click** - Open portal
- **Mobile** - Touch and drag to move, tap buttons to attack
